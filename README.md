## How to Execute the Programs

Follow the steps below to run any program from this repository:

Step 1: Clone the Repository

```bash
git clone https://github.com/pokharelayam1/DSA-Assignment.git
```
Step 2: Open a Program Directory

```bash
cd dsa_assignment/Prog_<number>
```
Step 3: Compile the Source Code
```bash
gcc Prog_<number>.c -o program
```
Step 4: Run the Executable
```bash
./program
```
